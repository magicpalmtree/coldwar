## Install

You'll need node.js version 4 or above.

```bash
git clone git@github.com:simonswain/coldwar.git
cd coldwar
cp config/index.sample.js config/index.js
npm install
node run

# http://localhost:4002
```

## Key Controls

* `tab` toggle options
* `esc` back to index
* `enter` restart scene
* `?` help.
* `\` diagnostics.
* `drag` pan
* `+`, `-`, '0' zoom
* `mousewheel` zoom.
